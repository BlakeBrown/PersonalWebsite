Hackroulette

Hackroulette is an intelligent chat roulette that analyzes a user’s twitter feed using machine learning algorithms, discovers their interests and strategically pairs users to create great conversations. I used Node.js and Socket.io to implement real time chat across multiple chat rooms, helped create the elegant front end and worked on integrating the API’s. Originally a 36-hour hackathon project, our team placed top 8 software hacks at HackPrinceton and were featured on one of the sponsor’s websites!

Technologies used: Indico and Twitter API’s, Node.js, Socket.io, HTML, CSS, Javascript
