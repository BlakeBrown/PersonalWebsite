Indoor Navigator

As part of my Engineering Design with Embedded Systems (ECE155) course last term, I was tasked with designing a unique pathfinding algorithm that takes users from their location to a chosen destination. Written in Android, the application uses real world map data to navigate the user around obstacles in their environment. In order for the indoor navigation to occur, I had to write several custom algorithms involving step counting and pathfinding from raw sensor input. This project was an amazing opportunity to learn more about mobile development as well as how to use graph traversal algorithms (Dijkstra's, A* search, Prim's, etc.) in modern applications.

Technologies used: Android, Java
