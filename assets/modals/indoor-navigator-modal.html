Indoor Navigator

As part of my Engineering Design with Embedded Systems course, I had to design a unique pathfinding algorithm to take users from their location to a chosen destination. Written in Android, the application uses real world map data to navigate a user around obstacles in their environment. I implemented a step counter with low-pass and high-pass acceleration filtering, a custom pathfinding algorithm and a UI for map navigation.

Technologies used: Android, Java, Graph Traversal Algorithms (Dijkstra's, A* search, Prim's, etc.)
